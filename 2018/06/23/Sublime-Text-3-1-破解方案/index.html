Sublime Text 3.1 3170 / 3176 注册码（附降级与禁止更新方法）

Sublime Text 3.1 更改了验证方法，之前所有的验证码都已失效，建议降级到3143 版本。

新增3.1 3176 可用注册码
此验证码为sublime text3 注册码，非sublime text2 注册码。

加入到hosts文件

  • Windows : c:/windows/system32/drivers/etc/hosts

  • Linux : /etc/hosts

  • Mac : /Private/etc
